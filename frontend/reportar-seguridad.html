Reportar Problema de Seguridad

Tu seguridad y privacidad son nuestra máxima prioridad. Reporta cualquier problema de seguridad que hayas identificado.

¿Qué puedes reportar aquí?

Este sistema está diseñado para reportar problemas relacionados con la seguridad informática, vulnerabilidades técnicas o brechas de seguridad en nuestros sistemas.

Ejemplos de problemas de seguridad:

  • Vulnerabilidades en el sitio web o aplicación móvil
  • Acceso no autorizado a cuentas o datos
  • Problemas con la autenticación o cifrado
  • Exposición de datos sensibles
  • Ataques de phishing usando la marca CEMI
¿Es una emergencia?

Si estás experimentando un problema de seguridad activo que requiere atención inmediata, contacta directamente a nuestro equipo de seguridad en seguridad@cemi.edu.ar o llama al 0800-123-CEMI (2364).
