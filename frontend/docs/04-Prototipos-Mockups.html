1. Introducción

Este documento presenta los prototipos de interfaz (wireframes y mockups) de las principales pantallas del sistema CEMI. Los diseños siguen los principios de usabilidad, accesibilidad y la identidad visual institucional.

Notas de Diseño:
  • Paleta de colores: Azul institucional (#1e3c72), Verde éxito (#2ecc71), Amarillo alerta (#f1c40f)
  • Tipografía: Inter (Google Fonts) para legibilidad
  • Diseño: Responsive, Mobile-First
  • Accesibilidad: Contraste WCAG AA, navegación por teclado
