Análisis de patrones

Nuestro sistema aprende tus ubicaciones habituales. Si accedes desde un lugar nuevo o lejano, se marca como inusual.

Usamos bases de datos de geolocalización IP para determinar ciudad y país aproximados. Puede tener variaciones de 20-50km.
