Compromiso con la inclusión digital

Nuestro Compromiso

En CEMI estamos comprometidos con garantizar que nuestra plataforma digital sea accesible para todas las personas, incluyendo aquellas con discapacidades visuales, auditivas, motoras o cognitivas.

Trabajamos continuamente para mejorar la experiencia de usuario y cumplir con los estándares internacionales de accesibilidad web, siguiendo las pautas WCAG 2.1 (Web Content Accessibility Guidelines).

Educación para todos

Creemos que la educación es un derecho universal y que las barreras tecnológicas no deben impedir el acceso al aprendizaje de idiomas.

Características de Accesibilidad

Nuestra plataforma incluye las siguientes características de accesibilidad:

Navegación por Teclado

  • Todos los elementos interactivos son accesibles mediante teclado
  • Uso de la tecla Tab para navegar entre elementos
  • Indicadores visuales claros de foco
  • Atajos de teclado para acciones frecuentes

Contenido Visual

  • Alto contraste entre texto y fondo
  • Texto redimensionable hasta 200% sin pérdida de funcionalidad
  • Alternativas textuales para imágenes (atributos alt)
  • No dependencia exclusiva del color para transmitir información

Contenido Multimedia

  • Subtítulos en contenido de video
  • Transcripciones de audio disponibles
  • Controles de reproducción accesibles
  • Opción para pausar animaciones automáticas

Formularios

  • Etiquetas claras para cada campo
  • Mensajes de error descriptivos
  • Instrucciones visibles antes de completar
  • Tiempo suficiente para completar formularios

Ayuda para la Navegación

Estructura del Sitio

Nuestro sitio utiliza una estructura clara con:

  • Encabezados jerárquicos (H1, H2, H3) para organizar el contenido
  • Navegación consistente en todas las páginas
  • Migas de pan para indicar ubicación
  • Mapa del sitio disponible en el Índice A-Z

Atajos de Teclado

Combinación Acción
Tab Navegar al siguiente elemento
Shift + Tab Navegar al elemento anterior
Enter Activar enlace o botón
Esc Cerrar menús o modales
Espacio Activar botones / Desplazar página

Compatibilidad con Tecnologías de Asistencia

Nuestra plataforma ha sido probada y es compatible con:

Lectores de Pantalla

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (macOS / iOS)
  • TalkBack (Android)

Navegadores

  • Google Chrome (últimas 3 versiones)
  • Mozilla Firefox (últimas 3 versiones)
  • Microsoft Edge (últimas 3 versiones)
  • Safari (últimas 2 versiones)

Herramientas de Magnificación

  • Lupa de Windows
  • Zoom de macOS
  • ZoomText

Reportar Problemas de Accesibilidad

Valoramos tu retroalimentación. Si encuentras alguna barrera de accesibilidad en nuestro sitio, por favor comunícate con nosotros:

Contáctanos

  • La tecnología de asistencia que utilizas
  • Una descripción del inconveniente
