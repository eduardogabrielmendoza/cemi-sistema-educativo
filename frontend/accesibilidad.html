Nuestro compromiso con la inclusión digital y la accesibilidad web para todos los usuarios.

Nuestro compromiso

En CEMI estamos comprometidos con garantizar que nuestra plataforma digital sea accesible para todas las personas, incluyendo aquellas con discapacidades visuales, auditivas, motoras o cognitivas.

Trabajamos continuamente para mejorar la experiencia de usuario y cumplir con los estándares internacionales de accesibilidad web, siguiendo las pautas WCAG 2.1 (Web Content Accessibility Guidelines).

Características de accesibilidad

Navegación por teclado

  • Todos los elementos interactivos son accesibles mediante teclado
  • Uso de la tecla Tab para navegar entre elementos
  • Indicadores visuales claros de foco
  • Atajos de teclado para acciones frecuentes

Contenido visual

  • Alto contraste entre texto y fondo
  • Texto redimensionable hasta 200% sin pérdida de funcionalidad
  • Alternativas textuales para imágenes (atributos alt)
  • No dependencia exclusiva del color para transmitir información

Contenido multimedia

  • Subtítulos en contenido de video
  • Transcripciones de audio disponibles
  • Controles de reproducción accesibles
  • Opción para pausar animaciones automáticas

Atajos de teclado

Combinación Acción
Tab Navegar al siguiente elemento
Shift + Tab Navegar al elemento anterior
Enter Activar elemento seleccionado
Escape Cerrar menús o modales
Ctrl + + Aumentar tamaño del texto
Ctrl + - Reducir tamaño del texto
Ctrl + 0 Restablecer tamaño del texto

Tecnologías asistivas compatibles

  • Lectores de pantalla: NVDA, JAWS, VoiceOver
  • Navegación por voz: Dragon NaturallySpeaking
  • Magnificadores: ZoomText, Windows Magnifier
  • Teclados alternativos: Teclados en pantalla, dispositivos de un solo switch

Compatibilidad con navegadores

Nuestra plataforma ha sido probada y es compatible con las versiones más recientes de:

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari (macOS e iOS)
  • Samsung Internet (Android)
